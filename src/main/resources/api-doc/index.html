https://{host}/api

  • host: required(string)

    The host according to the environment. [dev=byclistics.dev, prod=byclistics.de,local=localhost:8080]

Error handling

If an error occurs then they all are in the same format with the given error codes in the documentation.

Example

{
 "error": "error_code",
 "errorMessage": "human readable description"
}
